# Vinno Bot MVP

Минимальная версия:
- один источник: SimpleWine
- поиск через Playwright
- без LLM, чтобы было дёшево
- фото и цена берутся с карточки
- текст поста собирается из шаблона + локальный факт по сорту

## Команды
- `/start`
- `/post`

## Переменные
Смотри `.env.example`

## Railway
Нужен Docker deploy, потому что используется Playwright + Chromium.
